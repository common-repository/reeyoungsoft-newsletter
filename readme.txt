== Installation ==  

1. Upload the folder `reeyoungsoft-newsletter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit widgets area and drag and drop 'ReeyoungSoft Newsletter Widget' to widget area where you want to.
4. Enter your public & private API key.
5. Click 'Fetch available lists' to get your email lists then select one of them you want.
6. Select which fields (Email, First name etc) show or required when visitor subscribes. 
7. Click 'Save'.
8. Done!
